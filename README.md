# AbdOfficeHourNew
阿伯丁Office Hour预约系统开发，参考于教室预约系统（Manybits开发社）
## 项目结构：
1. cloudfunctions：用于编写云函数
2. miniprogram: 小程序主文件
  1. components：组件
  2. images: 图片
  3. pages：页面
     1. appointment：预约界面
     2. appointmentList：预约列表界面
     3. home：主页
     4. login：登录界面
     5. personal：个人信息界面
4. .gitignore 忽略git同步的文件
